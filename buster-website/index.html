Buster - Canvas-Rendered IDE

11 MB.

What Buster Does

Canvas-Rendered Editor: Every character is drawn on canvas. No DOM text nodes. Tree-sitter highlighting for 100+ languages. Code folding, minimap, AI ghost text.

Full Terminal Emulator: A real PTY-backed terminal rendered on canvas with sixel image support.

AI Agent: Chat with Claude, Ollama, Codex, or Gemini directly in the editor.

Git and GitHub: 30 built-in git commands plus GitHub PR and issue browsing.

Language Server Protocol: LSP support for Rust, TypeScript, Python, and Go. Rename, references, diagnostics panel.

Debugger: DAP-based debugging with breakpoints, stepping, stack frames, and variable inspection.

Quick Open and Command Palette: Cmd+P opens fuzzy file search.

Session Persistence: Auto-saves every 30 seconds.

WASM Extensions: Sandboxed extension runtime powered by Wasmtime with custom UI surfaces.

Panel Layouts: Seven layout modes.

11 MB Installer: Built on Tauri v2.

Building Extensions

Buster extensions are WASM modules that run in a sandboxed Wasmtime runtime.
